Instructions for Oura Ring API Token
To link data you need to get your personal API token and connect WSH to it. Get your personal API token by going here: https://cloud.ouraring.com/personal-access-tokens Instructions:
  1. Navigate to the Personal Access Tokens page https://cloud.ouraring.com/personal-access-tokens
  2. In the upper-right corner of the page click the "Create A New Personal Access Token" Button
  3. Enter a unique note for the new Personal Access Token you are about to generate
    1. This note is just a reminder for yourself about what the token is being used for
  4. Click "Create Personal Access Token" to submit the form and create your new Personal Access Token
  5. You should now have a new access token listed on the Personal Access Tokens page
