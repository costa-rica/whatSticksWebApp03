About Us

Mission

Easy to use, transparent, user feedback on their health data.

While always maintaining data control with the user.


Vision

What Sticks Health (WSH) is a health data storage infrastructure that provides personalized feedback. It is a work in progress. You can visit it at https://what-sticks-health.com to see what is there so far but here is a breakdown of what is going on. If you think you can help feel free to reach out.

Existing Platform

  • Data Storage: web application that allows users to upload Polar Beat training session, weight and user specified activity ( i.e. drinking coffee, headache, emotion, etc.,)
  • Feedback: Polar Beat training session indexed to model for cardio health
